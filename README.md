# Music-Catalog
A JavaFX-based music player app with MySQL database support.

## Features
- Play/pause songs
- View play history
- Add/delete songs
- Sort/filter by title

## How to Run
1. Install Java 17
2. Clone this repository
3. Open in IntelliJ
4. Run the `MusicCatalogue` main class

## Technologies Used

- JavaFX
- MySQL
- IntelliJ IDEA
